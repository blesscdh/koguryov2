koguryo V2 micromouse source    by Hwang Hayun .KOREA

Need CCS V3.3 (Code composer studio) for compile

koguryo micromouse's MCU is tms320f2808(TI).

cs_prj/ : ccs project folder for compile
si_prj/ : source insight project folder for Edit.

etc../ : program source


if you burn program through JTAG, you should modify "main/F2808.cmd"
